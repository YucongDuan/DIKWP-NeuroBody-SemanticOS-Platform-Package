# DIKWP NeuroBody SemanticOS Platform Package

A concrete platform delivery package for future brain/body-machine semantic interaction.

## Quick start
1. Open `prototype/index.html` in a browser.
2. Read `docs/01_DIKWP_NeuroBody_SemanticOS_Platform_Blueprint.docx`.
3. Use `DIKWP_NeuroBody_SemanticOS_Policy_Matrix.xlsx` to manage sensors, adapters, consent, risks, routes and benchmarks.
4. Review schemas under `schemas/` and API draft under `api/`.
5. Run `python src/neurobody_semanticos_demo.py` to generate a sample replay bundle.

## Safety boundary
This package is a research and product-design prototype. It is not a medical device, not a diagnostic or treatment system, and not an invasive BCI control protocol. All real human-data collection, clinical use, device control, or implanted/stimulation use requires independent ethics, clinical, regulatory and safety review.
